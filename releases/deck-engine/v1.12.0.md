# @deckio/deck-engine v1.12.0

## `deck:goTo` postMessage navigation

Added a `window.addEventListener('message')` listener in `SlideContext` that handles `deck:goTo` messages. This allows the launcher (or any parent frame) to programmatically navigate to a specific slide by posting:

```js
iframe.contentWindow.postMessage({ type: 'deck:goTo', project: '<id>', slideIndex: 3 }, '*')
```

Used by the launcher's slide thumbnail strip for click-to-navigate and by Puppeteer-based thumbnail generation.
