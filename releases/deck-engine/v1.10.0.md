# @deckio/deck-engine v1.10.0

**Released:** 2026-03-21

## What changed

### Added
- Add the `deck-port-powerpoint` Copilot skill for rebuilding PowerPoint and PDF decks as native DECKIO slides.

### Changed
- Stop provisioning the temporary `deck-generate-image` skill when syncing engine skills into deck projects.
- Surface the PowerPoint porting workflow in the engine-facing skill lists and scaffolded project guidance.

## Upgrade guide

Run `npx deck-init` in existing deck projects to copy the new skill set and remove temporarily disabled ones.
