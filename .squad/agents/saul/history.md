- **Shadcn Theme Rendering Bug Fixed (2026-03-14):** Three bugs caused all themes to look visually identical: (1) APP_JSX template never destructured `theme` from project config or passed it to `<SlideProvider>`, so `data-theme` always defaulted to "dark". (2) COVER_SLIDE_CSS template hardcoded dark-theme rgba values for meta bar (`rgba(22,27,34,0.8)`) and orb gradients — replaced with `var(--surface-overlay)` and `color-mix(in srgb, var(--token) N%, transparent)`. (3) No `color-scheme` CSS property on any theme — added `color-scheme: light` to shadcn/light, `color-scheme: dark` to dark. The actual CSS cascade was correct (unlayered theme `:root` properly overrides `@layer base` defaults), but the scaffolded project's hardcoded colors and missing theme prop made themes indistinguishable visually. Fixed in both engine source and test project. 136 tests pass.
